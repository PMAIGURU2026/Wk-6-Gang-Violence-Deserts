# Gang Violence & Resource Deserts Research Project

This research project analyzes the relationship between gang violence patterns and resource desert distribution.

## Project Overview

This project investigates how resource scarcity (food deserts, healthcare deserts, educational deserts, etc.) correlates with gang violence in urban areas.

## Files

- `fetch_data.py` - Script to pull live data from various sources (2-3 minutes)
- `requirements.txt` - Python dependencies
- `.gitignore` - Files to exclude from version control

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Run data collection
python fetch_data.py
```
